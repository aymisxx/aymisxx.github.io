MicroUAV-2D

An open-source ultra-lightweight 2D down-camera UAV simulation environment built for rapid movement-logic prototyping at fixed height. The platform provides discrete XY motion, clean field-of-view extraction, stable border handling, and a sharp separation between simulator core and autonomy logic, making it ideal for plug-and-play robotics experimentation without heavyweight dependencies.
