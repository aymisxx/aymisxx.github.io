DriftCTRL

A reinforcement learning project evaluating PPO, DQN with custom discretization, and A2C on the CarRacing-v3 environment. The study focuses on continuous versus discrete control, pixel-based visual input, training stability, reward evolution, and qualitative driving behavior, showing stable convergence for PPO while exposing the limitations of discretized and weakly convergent baselines.

PPO / DQN / A2C Visual RL Control Learning

Label Conditioned Robot Vision

A robotics-oriented generative vision study using conditional GANs to model semantic-controllable image synthesis. Framed as label or text conditioned generation for perception analysis, the project examines controllability, training stability, dataset scalability, and robotics-relevant failure modes across MNIST, Oxford-102 Flowers, and CUB-200 Birds rather than chasing photorealism for its own sake.

cGANs Generative Vision Semantic Conditioning
Open Repository

Perceiver Architecture Study

A minimal PyTorch reimplementation of the Perceiver architecture, centered on Fourier positional encodings, cross-attention from inputs into a learned latent bottleneck, and a latent Transformer stack. The project emphasizes clarity, architectural intuition, and reproducible training behavior on CIFAR-10 rather than brute-force scale, making the model anatomy easy to inspect and reason about.

Transformers PyTorch Architecture Study
Open Repository

ProphetLSTM GAN

An unsupervised anomaly detection framework for liquid rocket engine telemetry built around a hybrid LSTM-GAN design. The system combines reconstruction error, discriminator response, and trend-aware residual scoring to detect anomalies early, benchmark against traditional baselines, and study detection latency in a safety-critical engineering context where false calm can be expensive.

Anomaly Detection LSTM-GAN Telemetry AI
